## Architecture: Before vs After

````carousel
### ❌ Before (WebSocket Relay)
```
User Goal → WS → background.js → content_script.js → DOM
                                         ↕
                                  WebSocket relay
                                         ↕
                  FastAPI → ADK Agent → LLM → tool calls
                                         ↕
                                  WebSocket relay
                                         ↕
                               content_script.js → el.click()
```
Problems: Content script dies on navigation, fragile CSS selectors, no screenshots
<!-- slide -->
### ✅ After (Playwright Direct)
```
User Goal → FastAPI → Orchestrator
                          │
              ┌───────────┴───────────┐
              ▼                       ▼
        Playwright                ADK Agent
     (controls Chrome)         (plans actions)
              │                       │
   scraper.py │ action_handler.py     │ tools.py
              │                       │
              └───────────┬───────────┘
                          ▼
                    Chrome Browser
```
No content script needed. Playwright controls Chrome directly.
````

## Phase 3 Changes — What Was Rewired

### [tools.py](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/agent/tools.py)

| Before | After |
|---|---|
| `send_action_to_extension()` via WebSocket | `action_handler.click(page, ref)` via Playwright |
| Waited for `resolve_action_result()` from WS | Playwright returns result directly |
| [get_dom_snapshot()](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/agent/tools.py#171-210) requested via WS from extension | [scrape_page(page)](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/browser/scraper.py#214-295) via Playwright `page.evaluate()` |

New tool: **[select_option(selector, value)](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/agent/tools.py#97-110)** for dropdowns.

### [orchestrator.py](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/agent/orchestrator.py)

Key change: **scrapes the page via Playwright before sending goal to LLM.**

```python
# OLD: relied on extension to send DOM
page_context = payload.get("page_context")  # from WS message

# NEW: scrapes via Playwright directly
scraped = await scrape_page(page)
dom_text = format_elements_for_llm(scraped)
```

### [main.py](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/main.py)

**New endpoints:**

| Endpoint | Method | Purpose |
|---|---|---|
| `/api/run` | POST | Run automation headlessly (no extension needed) |
| `/api/connect` | POST | Connect to user's Chrome via CDP |
| `/api/launch` | POST | Launch a new headless browser |
| `/health` | GET | Check status + browser connection |

**Auto-connect:** When a WS message arrives and no browser is connected, [main.py](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/main.py) tries CDP first, then falls back to headless.

### [runner.py](file:///d:/Mini-proj/ghmc/Browser-automation-chatbot/web-extension/backend/agent/runner.py)

Simplified — removed [run_agent()](file:///D:/Mini-proj/GHMC_Demo%20_proper_working_v1/GHMC_Demo/ghmc-backend/agent/loop.py#524-776) and `format_dom_for_llm()` (now in orchestrator + scraper). ## How to Test Phase 3

### Option A: Headless API (no extension)
```powershell
# Start backend
uvicorn main:app --reload --port 8000

# Test via API
curl -X POST http://localhost:8000/api/run `
  -H "Content-Type: application/json" `
  -d '{"goal": "Search for hello world", "url": "https://google.com"}'
```

### Option B: CDP (connect to your Chrome)
```powershell
# Start Chrome with debugging
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

# Connect backend to it
curl -X POST http://localhost:8000/api/connect `
  -H "Content-Type: application/json" `
  -d '{"cdp_endpoint": "http://localhost:9222"}'

# Then run a task
curl -X POST http://localhost:8000/api/run `
  -H "Content-Type: application/json" `
  -d '{"goal": "Search for hello world on Google"}'
```
